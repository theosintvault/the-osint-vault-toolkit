# dork_patterns

Generates and manages Google Dork queries for advanced OSINT reconnaissance.

## Purpose

- Quickly generate search queries for Google, Bing, and other engines.
- Structure and manage reusable “dork” patterns for consistent investigations.

## Directory Structure

- `query_generator.py` - Uses provided keywords to generate search logic and patterns.

## Example Usage

```bash
python query_generator.py --input targets.txt
```
